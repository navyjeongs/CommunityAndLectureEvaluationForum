# CommunityAndLecture(자유게시판, 강의평가 게시판)
JSP를 이용한 자유게시판 및 강의평가 게시판 웹 사이트입니다.

<h2>Tomcat 및 MySQL 버전</h2>
Tomcat : v8.5
<br>
MySQL : 8.0.28
<br>
/WEBContent/WEB-INF/lib 위치에 자신의 MySQL 버전에 맞는 MySQL Connector 파일을 등록해야합니다.
